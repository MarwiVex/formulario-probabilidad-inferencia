# Formulario de Probabilidad e Inferencia Estadística

Este directorio contiene la versión en español del formulario, diseñado en LaTeX. Es un material de referencia ideal para estudiantes e investigadores en áreas de matemáticas, ingeniería y ciencia de datos. El documento abarca desde conceptos básicos de conteo hasta métodos avanzados de inferencia y procesos estocásticos.

---

## Contenido Principal

* **Fundamentos:** Combinatoria, Axiomas de Kolmogorov y Probabilidad Condicional.
* **Variables Aleatorias:** Distribuciones discretas y continuas, momentos y funciones generadoras.
* **Inferencia Estadística:** Estimación puntual (MLE, MoM), Intervalos de confianza y Pruebas de hipótesis.
* **Modelos Avanzados:** Regresión lineal y logística, Series de tiempo y Cadenas de Markov.
* **Guía Visual:** Mapa de relaciones entre distribuciones y comandos estadísticos básicos en R.

---

## Uso y Compilación

Si deseas compilar el documento PDF directamente desde el código fuente (`formulario_probabilidad.tex`), puedes utilizar tu entorno LaTeX preferido o ejecutar el siguiente comando en la terminal usando `pdflatex`:

```bash
pdflatex formulario_probabilidad.tex
```

*Nota: Dependiendo de tu configuración, es posible que necesites compilar el documento un par de veces para que las referencias internas y el formato se ajusten correctamente.*

# Probability and Statistical Inference Cheatsheet

This directory contains the English version of the cheatsheet, designed in LaTeX. It serves as an ideal reference material for students and researchers in mathematics, engineering, and data science. The document covers everything from basic counting concepts to advanced inference methods and stochastic processes.

---

## Core Content

* **Fundamentals:** Combinatorics, Kolmogorov Axioms, and Conditional Probability.
* **Random Variables:** Discrete and continuous distributions, moments, and generating functions.
* **Statistical Inference:** Point estimation (MLE, MoM), Confidence Intervals, and Hypothesis Testing.
* **Advanced Models:** Linear and logistic regression, Time Series, and Markov Chains.
* **Visual Guide:** Distribution relationship map and standard statistical commands in R.

---

## Usage and Compilation

If you wish to build the PDF document directly from the source code (`probability_cheatsheet.tex`), you can use your preferred LaTeX environment or run the following command in your terminal using `pdflatex`:

```bash
pdflatex probability_cheatsheet.tex
```

*Note: Depending on your setup, you may need to compile the document twice to ensure all cross-references and formatting are generated correctly.*
